# Nursing Clinical Insight

**AI-powered clinical analysis app for nurses and nursing students**

## Features

- Vital signs & symptoms input form
- AI-driven preliminary assessment (shock, sepsis, etc.)
- Risk level indicator (color badges)
- Suggested nursing actions
- Warning alerts & early warning prediction
- BMI Calculator, IV Drip Rate, Vital Signs Reference
- Bilingual: English / Arabic
- Responsive, hospital-friendly & accessible UI
- No patient data is stored (privacy-first)
